A simple and responsive social media profile page built using **HTML, CSS, and JavaScript**.  
This project demonstrates basic front-end design and interactive functionality.

## Features
- Profile picture, name, and bio
- Followers and following counters
- Follow/Unfollow button with dynamic counter update
- Clean and modern UI design
- Lightweight and beginner-friendly code

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
